# Site Photos

Drop your photos into the correct subfolder using the exact filenames below.
Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

Once added, run:
```
git add public/photos
git commit -m "add site photos"
git push
```
Vercel will deploy automatically with your photos.

---

## services/

Used on each individual service page — hero banner background AND the in-page content photo.

| Filename | Used on |
|---|---|
| `bookkeeping.jpg` | Bookkeeping service page |
| `payroll.jpg` | Payroll service page |
| `company-sec.jpg` | Company Secretarial service page |
| `permit-support.jpg` | Business Permit Support service page |

---

**Photo tips:**
- Landscape orientation works best (wider than tall)
- Minimum 1200px wide for hero images
- Hero images have a dark gradient overlay, so the subject should be clearly visible even at 50% brightness
- JPEGs under 500KB compress well without quality loss
